应用场景
=======
手机(andrioid)和pc(debian)通信有点不方便, 
很多时候群里面有小伙伴发一些学习链接或代码段等这些场景, 太长了, 照着屏幕一个一个字母敲, 麻烦费时又容易出错.
数据线都懒得连, 抽啊插的也麻烦
于是想到了用socket写个C/S通信脚本.
至少比一个个的敲方便点了.
